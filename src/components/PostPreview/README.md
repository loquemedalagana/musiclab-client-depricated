## post view
- 영상일 경우 썸네일 클릭 시 유튜브 페이지로 접속
- 서브 매뉴
  - [공통] go to post (포스트 디테일 페이지)
  - 유튜브일때랑, post일때랑 따로 설정
    - youtube: [go to post, add favorite]
    - post: [go to post, edit, delete]
- 하트: 좋아요, 채팅창: 댓글 로딩하기(view more -> detail page 연결)
